#### **1.3 创建标记代码映射表** ⭐⭐

**为什么:**
- EEG文件中的标记可能用代码表示 (S10, S14, etc)
- 需要将这些代码映射到标记名称 (problem, solution, etc)
- 这样才能自动验证

**创建文件:** marker_code_mapping.json
```json
{
  "eye_closed": {
    "possible_codes": ["EC", "S0", "EyeClosed"],
    "description": "眼睛闭合基线",
    "expected_count": 1
  },
  "1_problem": {
    "possible_codes": ["S10", "PROB", "Problem1"],
    "description": "问题1呈现",
    "expected_count": 1
  }
}
```

**预期工作量:** 1-2 小时
